如果您想自定义一个分类，请将如下结构的代码添加到 /source/vendor/menus/file.js的 tech_file_list 变量中
type 为您自定义的分类
sign 为分类标识
themes为您分享的主题，可以有多个，若有多了，则themes数组的长度则大于1
themes中的 theme 为当前分享的主题 file数组为 此次分享用到的文件。
接下来，将您的文件放到 downlaod文件夹下，并将绝对路径添加到 address上。
{
    type:'前端',
    sign:'fe',
    themes:[
      {
        theme:'网站分析的实现',
        file:[
          {
            name:'名称1',
            address:'/source/download/test.zip'
          }
        ]
      }
    ]
}

如果当前分类有您所需要的应用，那么您只需要在 当前分类下的themes添加如下格式的代码就可以了
{
	theme:'网站分析的实现',
	file:[
	  {
		name:'名称1',
		address:'/source/download/test.zip'
	  }
	]
}

如果您只是想在原有的基础上增加一个文件的话，那么只需要写如下格式的代码放到对应的主题下的 file数组里边就可以
{
	name:'名称1',
	address:'/source/download/test.zip'
}

添加时请注意json格式，如果接送格式错误，可能导致页面无法正常显示哦
